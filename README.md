## Installation

```bash
npm install -g discogs-crate-digger
```

## Usage

When using the command it will trigger an oauth flow with spotify asking for access to create a playlist on your behalf

```bash
discogs-crate-digger <username>
```

or

```bash
dcd <username>
```
